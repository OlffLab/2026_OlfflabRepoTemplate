# how to view these documentation files  

These documentation files are formatted in Markdown (.md) format. This is a plain text with some simple formatting codes, that can be read by a viewer to show formatting, such as headers, pieces of scripts, and links
To view these .md files in their formatted way , right-click one of the .md files in the Positron explorer (left panel) and select "Open Preview".  

It is recommended to not use the "preview" function in Positron (at the top of this window) as it creates a lot of unnesscary files (as .html)
